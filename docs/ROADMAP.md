# Roadmap

## Near term (0-3 months)

- tighten preset consistency across remaining shared UI wording
- improve demo reliability and first-run setup ergonomics
- strengthen release automation and publication hygiene
- expand policy/reporting docs and operator guidance

## Mid term (3-9 months)

- enterprise identity integration (SSO/SAML planning and implementation)
- richer policy authoring workflows and approval lifecycle
- stronger compliance export workflows and audit evidence packaging
- enhanced provider controls and reliability telemetry

## Longer term (9+ months)

- advanced multi-region deployment and resilience hardening
- broader governance automation for large multi-org operations
- additional vertical presets and ecosystem integrations
- deeper security posture controls for high-assurance environments

## Roadmap guardrails

- preserve Sentinel as shared platform core
- preserve SentinelLaw as intentional legal preset
- avoid feature expansion that weakens governance-first architecture
